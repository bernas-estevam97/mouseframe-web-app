# Mouseframe



## Information

This is an application to automatically return distance values from various measurements of __mus musculus__ walking behaviour. This application's purpose resides in simply saving more than time from researchers that needs these  walking behaviour measurements for their studies. The data obtained is easily exported to excel tables with all the proper indentification done.  


## Description
Mouseframe accepts image filetypes from the user with the marked footprints of the animals (usually red to represent the front legs and blue to the hind legs). These images are scanned by printers. The user will need to mark these same footprints with the respective points provided by the application various functions. In order to obtain values in cm it is required by the user to present in the scanned images a line measurement of 1 cm in the scanned images. This measurement is then calculated by another function which will return the ammout of pixels 1 cm will have.

When all the footprints are marked and the pixels in 1 cm is measured, the user can generate a table with all the measurements which will show the respective points and the measurement type performed for each calculation. Plus the user can generate a table that includes mean values for every measurement type.
