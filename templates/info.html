    • Purpose of MouseFrame

    • This application automates the measurement and calculation process traditionally performed manually in laboratory settings, reducing time and minimizing human error. It uses digitized images of mouse track sheets as the basis for analysis.
    • The application provides an intuitive interface that processes scanned sheets quickly, allowing results to be exported to Excel spreadsheets or copied directly to the clipboard for further analysis.
    • MouseFrame converts scanned images into an interactive canvas, enabling users to annotate the images, perform automated measurements, and generate accurate calculations efficiently.

    Information on mice walking behaviour analysis

    The front paws of the mice are marked in red and the hind paws in blue. This analysis goes through 8 different types of measurements that are all manually calculated in a laboratory setting. The 8 measures are the following:

    • Stride length left front
    • Stride length left hind
    • Stride length right front
    • Stride length right hind
    • Overlap Left
    • Overlap Right
    • Stride Width Front
    • Stride Width Hind

    How are the values obtained?

    All measurements are calculated using the Euclidean distance formula between points in a two-dimensional plane. These calculations rely on the user accurately identifying and labeling each paw print according to its position (front or rear, left or right). The application processes the footprints sequentially, performing the required calculations for each set based on their recorded order. Maintaining the correct footprint sequence is essential, as altering the order may produce inaccurate measurements. Understanding this workflow helps ensure reliable and consistent results.

    • Stride Length – Left Front: Calculates the Euclidean distance between consecutive left front (red) footprints, measuring the distance from L(x) to L(x+1). The final footprint in each sequence is excluded from the calculation.
    • Stride Length – Left Hind: Calculates the Euclidean distance between consecutive left hind (blue) footprints, measuring the distance from L(x) to L(x+1). The final footprint in each sequence is excluded from the calculation.
    • Stride Length – Right Front: Calculates the Euclidean distance between consecutive right front (red) footprints, measuring the distance from R(x) to R(x+1). The final footprint in each sequence is excluded from the calculation.
    • Stride Length – Right Hind: Calculates the Euclidean distance between consecutive right hind (blue) footprints, measuring the distance from R(x) to R(x+1). The final footprint in each sequence is excluded from the calculation.
    • Overlap – Left: Measures the Euclidean distance between the corresponding left front (red) and left hind (blue) footprints that share the same index.
    • Overlap – Right: Measures the Euclidean distance between the corresponding right front (red) and right hind (blue) footprints that share the same index.
    • Stride Width – Front: Calculates the front stride width using the front (red) footprints. The user must specify whether the first footprint in the sequence is on the left or right. The measurement is obtained by calculating the distance between the selected footprint and an intermediate reference point, which shares the same x-coordinate as the selected footprint and the average y-coordinate of the corresponding left and right front footprints.     • Stride Width – Hind: Calculates the hind stride width using the hind (blue) footprints. As with the front stride width, the user must specify whether the first footprint in the sequence is on the left or right. The measurement is obtained by calculating the distance between the selected footprint and an intermediate reference point, which shares the same x-coordinate as the selected footprint and the average y-coordinate of the corresponding left and right hind footprints.
