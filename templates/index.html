Image Analysis - Mus Musculus walking behaviour

Information and requirements

  • Only image file formats are supported.
  • Make sure you note which dpi was used in the case of scanned images to later add the number to further calculus in order to increase the results precision.
  • In order to draw dots to perform the calculations, this is done in canvas. The canvas as the name implies if a surface where the user can draw, in this case only dots of different sizes and colors can be added.
  • The canvas will be automatically set to the uploaded image dimensions although a canvas bigger than the uploaded image would not make a difference in the distance calculations as long as the user adds the dots exclusively in the image.

Canvas starts with same size as the image uploaded. Change below if necessary.



Canvas size is set to default (300x150).



Default point radius size is set to 4px
