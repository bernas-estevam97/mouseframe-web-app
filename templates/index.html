Image Analysis - Mus Musculus walking behaviour

Information and requirements

  • Only image file formats are supported.
  • Make sure you note which dpi was used in the case of scanned images to later add the number to further calculations in order to increase the results precision.
  • In order to draw dots to perform the calculations, this is done in canvas. The canvas as the name implies is a surface where the user can draw, in this case only dots of different sizes and colors can be added.
  • The canvas will be automatically set to the uploaded image dimensions although a canvas bigger than the uploaded image would not make a difference in the distance calculations as long as the user adds the dots wherever it is wanted.
  • Make sure your image resolution fits your computer screen as the canvas will automatically adopt its dimensions.
  • All the functionalities are shown after successful image upload.
