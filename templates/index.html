MOUSE FRAME - mus musculus walking behaviour analysis tool

Notices and updates - Version 0.8.2

  • Fixed visual bugs for most resolutions.
  • Added compatibility for most resolutions but this app is not a mobile app. Don't expect it to function on mobile phones (may work on tablets with big resolutions).
  • Added function to save values and use them without app refresh, this way not restarting your progress.
  • The saved values require an image size of which the measurement was saved in order to keep track of right cm to pixel conversions and avoid measurement usage in wrong resolutions.

Brief information and requirements

  • Only image file formats are supported (.tiff formats are not yet supported).
  • In order to draw dots to perform the calculations, this is done in canvas. The canvas as the name implies is a surface where the user can draw, in this case only dots of different sizes and colors can be added.
  • All the functionalities are shown after successful image upload.
  • For more information on this application and to see how the calculations are made for each measurement, check this link.

Instructions for manual removal:

  • Your points appear below as you add them.
  • Select your desired point and two options for removal will appear:
    • Remove: Delete the point and update the count without the removed point.

Help - What does Add Entries option do?

  • ➣   This function allows users to calculate distances between the points marked above.
  • ➣   In contrast with the "Add entries with average" function, this option does not require the pixels in 1 cm measurement (if the value is available it will still convert px to cm). If the user only wants to know the distance in pixels this option may suffice.
  • ➣   This function does not require a full set of marked dots to perform all available measurements, but it will throw an alert that some measurements could not be measured for the lack of certain dots.
  • ➣   Overall, this measurement type is less sensitive and great for when the user only wants a few set of measurements and not all.

Help - What does Add Entries with Average do?

  • ➢   This function calculates a complete set of measurements, converts the values to cm and displays mean values for each measurement in cm.
  • ➢   "Add entries with average" requires the px in 1 cm measurement beforehand.
  • ➢   It also needs all marked dots in the correct spots to perform all the available calculations.
  • ➢   This method/function is the most complete and sensitive when it comes to data retrieval.
